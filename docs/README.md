# Home Sweet Home

Aplicación personal para la gestión integral del hogar: gastos, compras y almacén. Proyecto estructurado de forma profesional para facilitar el mantenimiento y la posible ampliación.

- **Frontend:** React
- **Backend:** Java (Spring Boot)
- **Base de datos:** H2 (desarrollo), PostgreSQL (producción)
- **Objetivo:** Control sencillo y completo de los gastos y recursos de los hogares.

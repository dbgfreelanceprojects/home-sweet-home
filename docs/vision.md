# Visión General

Home Sweet Home centraliza la gestión del hogar, permitiéndote registrar y analizar gastos, administrar productos en el almacén y gestionar compras desde móvil y PC.

- Proyecto personal, pensado con calidad y profesionalidad.
- Base para futuras ampliaciones: control de tareas, recetas, alertas, etc.
- Enfoque multiusuario y multihogar para mayor flexibilidad.

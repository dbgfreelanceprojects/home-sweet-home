# Manual de Usuario

## Primeros pasos

1. Accede a la aplicación y regístrate usando tu email.
2. Inicia sesión.
3. Crea tu primer hogar.
4. Añade tus gastos iniciales (luz, agua…).
5. Añade productos al almacén y registra compras.
6. Consulta y filtra información por hogar.

## Futuras mejoras

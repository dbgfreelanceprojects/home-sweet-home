# Arquitectura Técnica

## Componentes

- **Frontend:** Aplicación SPA en React.
- **Backend:** API RESTful en Java Spring Boot.
- **Base de datos:** relacional.
- **Seguridad:** Autenticación JWT.
- **Despliegue:** Heroku (backend), Vercel/Netlify/Azure Static Web Apps (frontend).

## Comunicación

- HTTP con formato JSON entre frontend y backend.
- Gestión de sesión basada en JWT enviado como header.

## Estructura básica

- `/frontend`: código React
- `/backend`: código Java Spring Boot
